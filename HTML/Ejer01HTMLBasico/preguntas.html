1)Dar un par de ejemplos de uso de etiquetas meta.

la etiqueta meta se usa en todo archivo html para declarar configuraciones respecto al mismo, como linkeos a paginas de estilo css, fuentes de google, que tipo de charset etc.

2)Nombrar al menos 2 atributos aplicables a la etiqueta HTML

lang que hace referencia al lenguaje, y xmlns que especifica el espacio de nombres xml del documento

3)Como se relaciona un header http con la etiqueta head de HTML?

mediante la linea "meta http-equiv="X-UA-Compatible" content="IE=edge""

4)Que es UTF8?

Es un formato de codificación de caracteres Unicode e ISO 10646 que utiliza símbolos de longitud variable.

5)Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?

el atributo style, y es aplicable a todas las etiquetas que son componentes dentro de una vista

6)Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

target="_blank" en la etiqueta de enlace

7)Explicar cuando usarían etiquetas semánticas y con que propósito.

Las etiquetas semanticas se deben utilizar siempre para sectorizar un archivo html y asi estar mas organizado. ademas permiten que los navegadores indexen mejor la pagina.

8)¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.

width y height para darle tamaño deseado. rowspan y colspan dependiendo del tipo de tabla que se quiera hacer para sectorizar la tabla, border para aplicar distintos atributos al borde.

9)¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

la etiqueta body de un documento http contiene todo el contenido html de la vista, es decir las etiquetas html y su contenido, mientras que el body de http puede variar

10)¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

colspan y rowspan y se aplican al elemento td
